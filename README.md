# Overview

Everoute ipam 是用于 CNI 的 IPAM.

# 功能介绍
- 支持用户通过 IP 池动态设置可分配 Pod block 的 IPAM
- 支持 Pod 指定为其分配 IP 的 IP 池
- 支持 Pod 指定 IP。
